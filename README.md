## Installation

- Clone and open project in editor

- run "yarn" or "npm install"

- rename sample.env to .env.local

- create API key, instructions [here][df]

- paste API key in .env.local

- run "yarn dev" or "npm run dev"

Take into accound, that max limit of requests per minute is 10, so sometimes you won`t get any data because of this
